Prompt Pronto · Planilha de Notas

Criando uma planilha de notas com IA

Claude · Prompt
// Cole no Claude (claude.ai) e ajuste os campos em amarelo

Crie uma estrutura de planilha para controle de notas de uma turma
com [número de alunos] alunos do [ano/série].

A planilha deve ter:
- Colunas: Nome do aluno, [atividade 1], [atividade 2], Prova, Média
- Pesos: [atividades = X%], [prova = Y%]
- Coluna de situação: Aprovado / Recuperação / Reprovado
- Fórmula da média ponderada em Excel/Google Sheets

Explique cada fórmula usada de forma simples.
💡 Depois peça: "Agora gere também uma fórmula para calcular a média da turma e destacar alunos abaixo de 5."

Fórmulas com IA

Peça fórmulas em linguagem natural

Média ponderada
Calcular média com pesos diferentes para atividades e prova.
=B2*0.2 + C2*0.2 + D2*0.6
Situação do aluno
Mostrar "Aprovado", "Recuperação" ou "Reprovado" automaticamente.
=SE(E2>=7,"Aprovado",SE(E2>=5,"Recuperação","Reprovado"))
Contar aprovados
Contar quantos alunos estão aprovados na turma.
=CONT.SE(F2:F35,"Aprovado")
Média da turma
Calcular a média geral de toda a coluna de médias.
=MÉDIA(E2:E35)
✦ Basta descrever o que você quer em português — o Claude escreve a fórmula correta para Excel ou Google Sheets.

Prompt Pronto · Análise de Turma

Pedindo análise de desempenho ao Claude

Claude · Análise de dados
// Cole suas notas reais no lugar dos exemplos

Aqui estão as médias do [bimestre] da turma [turma]:
[Liste: Nome — Nota]
Ex: Ana 7.5 / Bruno 4.0 / Carla 8.2 ...

Com base nesses dados:
1. Quantos alunos estão abaixo da média?
2. Qual é a média geral da turma?
3. Quais alunos precisam de mais atenção?
4. Sugira uma estratégia de intervenção simples.
💡 Sem precisar de Excel — cole os dados no chat e receba a análise em segundos.
Boas Práticas

Como usar o Claude com planilhas

✔ Faça isso
  • Descreva o que você quer em linguagem simples
  • Peça fórmulas para Excel E para Google Sheets
  • Cole dados sem nomes reais — use "Aluno 1, Aluno 2"
  • Peça ao Claude para explicar cada fórmula
  • Use o Claude para interpretar resultados, não só calculá-los
✖ Evite isso
  • Inserir nomes e dados pessoais reais de alunos
  • Usar fórmulas sem testar em uma planilha real
  • Depender da IA para decisões pedagógicas importantes
  • Esquecer de verificar se os cálculos estão corretos
  • Usar formatação de fórmulas sem ajustar para o idioma (ponto x vírgula)
